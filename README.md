# the-distance-of-a-smile
The Distance of a Smile 

The Distance of a Smile is an immersive 2D animated mobile game featuring high-quality anime-style character designs and a compelling narrative. 

Overview: 

The Distance of a Smile is an anime-style interactive mobile game where player choices shape relationships, emotions, and the ending of the story. 

You play as a young student navigating friendship, love, and competition while facing a mysterious rival and discovering the meaning behind a simple smile. 

 

Key Features: 

Stunning 2D Art: High-quality anime illustrations featuring detailed streetwear and tech-wear designs. 

Dynamic Characters: Interact with a diverse cast, from the cool "street-style" heroine to the sophisticated "rich-rival" elites. 

Multilingual Support: Fully localized experience via translations.js. 

Optimized for Mobile: Smooth performance and intuitive UI built with web technologies (HTML5, CSS3, JavaScript). 

 

Characters: 

Main Character: A sharp, dark-haired youth in a classic suit. 

The Partner: Warm-hearted companion who changes your journey. 

Rich Rival – Charismatic competitor with hidden motives, a group of handsome, high-society challengers. 

 

Technical Stack: 

Frontend: HTML5, CSS3 (for styling and layout). 

Logic: JavaScript (game.js) for core mechanics and state management. 

Assets: Transparent PNG character renders for modular UI placement. 

 

How to Run: 

Clone the repository: git clone https://github.com/subhadipdas-07/the-distance-of-a-smile.git 

Open index.html in your browser. 

 

Project Structure 

index.html      → Game interface 
style.css       → Game styling 
game.js         → Game logic 
translations.js → Dialogue system 
assets/         → Characters & images
